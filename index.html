Basic Task

Point-In-Polygon

In searching for a reliable PIP (Point-In-Polygon) algorithm, we found many solutions. The one that we found was easiest to code was the following (created by [180188381]):

x <- c(1,3,4.5,3)
y <- c(1,2,1,0.5)
p <- c(3,1)

norm_vec <- function(x) sqrt(sum(x^2))
dot <- function(x,y) sum(x*y)
angle <- function(x,y){
  theta <- acos(dot(x,y)/(norm_vec(x)*norm_vec(y)))
  return(theta)
}

in_or_out <- function(x,y,p){
  total_angle <- 0
  for (i in 1:length(x)){
    if (i == length(x)){
      vector1 <- c(x[i] - p[1], y[i] - p[2])
      vector2 <- c(x[1] - p[1], y[1] - p[2])
      total_angle <- total_angle + angle(vector1, vector2)
    } else{
      vector1 <- c(x[i] - p[1], y[i] - p[2])
      vector2 <- c(x[i+1] - p[1], y[i+1] - p[2])
      total_angle <- total_angle + angle(vector1, vector2)
    }
  }
  if (round(total_angle, 5) == round(2*pi,5)){
    return(TRUE)
  } else{
    return(FALSE)
  }
}

in_or_out(x,y,p)
## [1] TRUE

This program is checking if the point \((3,1)\) is inside or outside a polygon with the given \(x\) and \(y\) co-ordinates given in lines 1 and 2 respectively. The ‘checking’ process creates lines between the point and each vertex of the polygon, and sums the angles between each pair of vertices. If the angles add up to \(2\pi\), then the point is inside the polygon. If the angles add up to anything else, the points are outside.
